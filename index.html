О проекте F-Base

F-Base — это мини-фреймворк в стиле Material Design для вёрстки одно страничных веб-сайтов, которые, в основном, содержат некую информацию (changelog, инструкции и т.д).

Также F-Base может использоваться в приложениях-учебниках (как DevPlus). Загрузить тестовое приложение с интеграцией с F-Base можно по ссылке.

Разработка F-Base

На данном этапе разработки, F-Base имеет весьма скудный набор стилей. Разработка ведётся открыто на Github. Вы тоже можете подключиться к разработке.

Компоненты F-Base UI

Чтобы ознакомить Вас с интерфейсом F-Base необходимо показать компоненты этого интерфейса.

Компонент Alert

Alert - это весьма необходимый элемент интерфейса. Alert выполняет роль уведомления-подсказки. Выглядит это примерно так:

Привет!

Это Alert из F-Base.

Чтобы разместить Alert у себя на разметке используйте контейнер <div></div> с классом f-alert:

<div class="f-alert">
  <h4>Привет!</h4>
  <p>Это Alert из F-Base.</p>
</div>

Конечно, Alert имеет разновидности. Чтобы вывести на экран сообщение об опасности воспользуйтесь этим Alert:

Привет!

Это Alert Danger из F-Base.

<div class="f-alert f-alert-danger">
<!-- Тут просто добавили класс f-alert-danger -->
</div>

Чтобы вывести на экран информативное сообщение воспользуйтесь этим Alert:

</div>

Компонент Page Header

Page Header - это элемент, который является заголовком страницы (точно как на этой) и распологается в самом начале страницы.

Чтобы вывести на экран такой заголовок, необходимо использовать тег <div></div> с классом f-page-header:

<div class="f-page-header">
  <h1>Title</h1>
  <p>Subtitle</p>
</div>

Но запомните, что в Page Header есть заголовок <h1><h1/> и подзаголовок <p></p>. Использовать другие теги не стоит. Будут искажения интерфейса.
